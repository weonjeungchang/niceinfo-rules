# 빠른 시작 가이드 🚀

NICE평가정보 내규 챗봇을 빠르게 시작하는 방법입니다.

## 준비 사항

- Python 3.8 이상
- OpenAI API 키
- `./reference` 폴더에 내규 문서

## 3단계로 시작하기

### 1단계: 설치 (2분)

```bash
# 의존성 설치
pip install -r requirements.txt

# 환경 변수 설정
copy env_template.txt .env  # Windows
# 또는
cp env_template.txt .env    # Linux/Mac

# .env 파일 편집하여 API 키 입력
# OPENAI_API_KEY=your-api-key-here
```

### 2단계: 시스템 체크 (1분)

```bash
python check_system.py
```

모든 항목이 ✓ 표시되는지 확인합니다.

### 3단계: 실행 (5-10분)

```bash
# 방법 A: 자동 설정 (권장)
streamlit run app.py

# 방법 B: 수동 설정
python setup_db.py    # 벡터 DB 생성 (첫 실행 시만)
streamlit run app.py  # 챗봇 실행
```

첫 실행 시 문서 인덱싱에 5-10분 정도 소요됩니다.

## 주요 기능

### 참고 문서 확인

각 답변 아래의 "📄 참고 문서 보기"를 클릭하면:
- 답변의 근거가 된 문서 파일명
- 문서 카테고리
- 내용 미리보기

를 확인할 수 있습니다.

### 대화 내역 관리

사이드바에서:
- **대화 내역 지우기**: 새로운 주제로 대화 시작
- **참고 문서 표시**: 출처 정보 표시/숨김

### 문서 업데이트

문서가 변경되었을 때:
1. 새 문서를 `./reference` 폴더에 추가
2. 사이드바의 "🔄 문서 재인덱싱" 클릭
3. 5-10분 대기
4. 완료!

## 문제 해결

### API 키 오류

```
❌ OPENAI_API_KEY가 설정되지 않았습니다
```

해결:
1. `.env` 파일이 있는지 확인
2. 파일 내용에 `OPENAI_API_KEY=sk-...` 형식으로 입력되어 있는지 확인
3. 애플리케이션 재시작

### 문서 로딩 실패

```
❌ 로드된 문서가 없습니다
```

해결:
1. `./reference` 폴더 확인
2. 지원 형식(.doc, .docx, .xlsx, .pdf) 확인
3. 파일이 손상되지 않았는지 확인

### 느린 응답

**정상적인 경우:**
- 첫 질문: 3-5초 (벡터 DB 로드)
- 이후 질문: 2-3초 (OpenAI API 호출)

**너무 느린 경우:**
- 네트워크 연결 확인
- OpenAI API 상태 확인 (status.openai.com)
