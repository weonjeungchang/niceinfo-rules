## 기본 질문 예제

### 예제 1: 복무 규정 조회

**질문:**
```
직원의 근무시간은 어떻게 되나요?
```

**예상 답변:**
```
NICE평가정보의 복무규정에 따르면, 직원의 근무시간은 다음과 같습니다:

- 근무 시작: 오전 9시
- 근무 종료: 오후 6시
- 점심시간: 오후 12시 ~ 오후 1시

[참고 문서: 복무규정_20250223시행.doc]
```

### 예제 2: 연차 휴가 문의

**질문:**
```
연차 휴가는 연간 몇 일이며, 어떻게 사용하나요?
```

**예상 답변:**
```
연차 휴가에 관한 규정은 다음과 같습니다:

1. 연차 휴가 일수:
   - 1년 근속: 15일
   - 3년 이상 근속 시 2년마다 1일 추가 (최대 25일)

2. 사용 방법:
   - 인사시스템을 통해 신청
   - 최소 3일 전 신청 원칙
   - 부서장 승인 필요

[참고 문서: 인사규정_20230809.doc]
```

## 효과적인 질문 팁 💡

### ✅ 좋은 질문

1. **구체적이고 명확한 질문**
   ```
   ❌ 휴가는?
   ✅ 연차 휴가 사용 절차와 신청 기한을 알려주세요
   ```

2. **맥락이 있는 질문**
   ```
   ❌ 얼마?
   ✅ 결혼 경조사 지원금은 얼마인가요?
   ```

3. **규정 이름 언급**
   ```
   ❌ 회사 규칙 알려줘
   ✅ 복무규정에서 지각 관련 내용을 알려주세요
   ```

### ❌ 피해야 할 질문

1. **너무 모호한 질문**
   ```
   ❌ 회사 규정
   ❌ 이것 저것
   ```

2. **문서 범위 밖 질문**
   ```
   ❌ 경쟁사 정보
   ❌ 일반 상식
   ```

3. **개인 정보 요청**
   ```
   ❌ 특정 직원 정보
   ❌ 급여 세부 내역
   ```

## 대화 시나리오 예제

### 시나리오 1: 신입사원 온보딩

```
사용자: 안녕하세요, 신입사원입니다. 출근 시간이 궁금합니다.
챗봇: 안녕하세요! 복무규정에 따르면 출근 시간은 오전 9시입니다...

사용자: 지각하면 어떻게 되나요?
챗봇: 지각에 관한 규정은 다음과 같습니다...

사용자: 연차는 언제부터 사용할 수 있나요?
챗봇: 신입사원의 연차 사용은...
```

### 시나리오 2: 휴가 신청

```
사용자: 다음 주에 휴가를 가고 싶은데 어떻게 신청하나요?
챗봇: 휴가 신청 절차는 다음과 같습니다...

사용자: 반차도 가능한가요?
챗봇: 네, 반차 사용이 가능합니다. 관련 규정은...
```

### 시나리오 3: 경조사
